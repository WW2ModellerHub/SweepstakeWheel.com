Random Name Picker FAQ

How do I pick multiple names fairly?

Turn on remove winner mode. After each spin, the winning name is removed from the wheel so they cannot be picked again.

Can teachers use this as a random student picker?

Yes. Add student names, load the wheel and use remove winner mode for questions, presentations or classroom turns.

Can I use it for raffles or meetings?

Yes. It works for raffles, team draws, meeting order, chores and any list of people.

Are names uploaded anywhere?

No. Names run locally in the browser and are not uploaded by this tool.

For teachers and classrooms

The Random Name Picker is built around the classroom use case. Paste a class list, keep remove-winner mode switched on, and the wheel removes each selected student after the spin. That means you can choose students for questions, reading turns, presentations or group tasks without the same name coming up repeatedly.

It also works well for supply teachers, starter questions, plenary tasks, house points and choosing who presents first. Because the names stay in the browser, you can use it quickly without uploading a class list to an account.

Remove after pick mode

Remove after pick is the key difference between a normal wheel and a useful name picker. A standard wheel can pick the same person twice. This page can remove the winner from the remaining list so every round stays fair until the list runs out.
